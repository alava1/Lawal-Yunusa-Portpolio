Portfolio Projects

1. Network Traffic Analysis

This project focuses on analyzing network traffic to understand communication patterns, identify protocols, and detect suspicious behavior within packet captures. Using Wireshark, I review traffic flows, inspect packet details, and document observations related to HTTP, DNS, TCP, and other common protocols. The objective of this project is to strengthen my understanding of network visibility and traffic-based threat analysis.

Skills demonstrated

- Packet analysis
- Protocol inspection
- Traffic investigation
- Security observation and reporting

---

2. Vulnerability Assessment Lab

This project demonstrates a structured vulnerability assessment process in a controlled lab environment. It includes host discovery, service enumeration, port scanning, and the identification of possible security weaknesses using tools such as Nmap and vulnerability assessment platforms. The project also includes a written summary of findings and recommendations for reducing risk.

Skills demonstrated

- Vulnerability assessment
- Port scanning
- Enumeration
- Security analysis
- Remediation planning

---

3. Web Application Security Testing

This project is focused on testing intentionally vulnerable web applications in a safe lab environment in order to understand common web security issues. It includes using web security tools to inspect requests, identify weaknesses, and document findings related to insecure application behavior. The goal is to build practical awareness of web vulnerabilities and improve security testing methodology.

Skills demonstrated

- Web security testing
- Request/response analysis
- Vulnerability identification
- Security reporting

---

4. Linux Hardening Project

This project demonstrates practical Linux system security improvements through hardening activities such as reviewing services, improving permissions, strengthening user access, and applying basic system security configurations. The aim is to understand how operating system configurations affect security posture and how defensive changes can reduce exposure.

Skills demonstrated

- Linux administration basics
- Security hardening
- Access control awareness
- System configuration review

---

5. SIEM and Log Analysis

This project focuses on reviewing logs and security-related events to identify suspicious patterns, failed logins, abnormal activity, or indicators of compromise. It reflects an entry-level understanding of security monitoring and the role of log analysis in a Security Operations Center (SOC) environment.

Skills demonstrated

- Log analysis
- Event review
- Monitoring concepts
- Security investigation basics
